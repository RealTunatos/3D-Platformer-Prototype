# 3D Platformer Prototype (Unreal Engine 5.6)

My first Unreal Engine 5 project. This is for learning Blueprint programming and core gameplay systems.



##Features

- Sprint
- Dash
- Double Jump
- Checkpoint System
- Death Zone (Lava)
- Timer
- HUD
- Main Menu
- Pause Menu
- Restart System
- Level Complete Screen
- Moving Platforms
- Crushing Cubes
- Sound Effects


## Controls

| Key | Action |
|------|--------|
| W A S D | Move |
| Space | Jump |
| Left Shift | Sprint |
| E | Dash |
| G | Interact |
| Esc | Pause |


## Built With

- Unreal Engine 5.6
- Blueprints


## Purpose

This project is made for learning and practicing the fundamentals of Unreal Engine gameplay programming with Blueprints.

The project contains basic gameplay mechanics and very few assets. The focus was on implementing gameplay systems rather than creating polished visuals.

## What I Learned

- Blueprint scripting
- Character movement mechanics
- State management
- UI (HUD, Main Menu, Pause Menu)
- GameMode and Player Controller
- Checkpoint and respawn systems
- Widget communication
- Basic game flow
- Audio implementation
